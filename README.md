# ChipMap
A simple tool to visualize chip layouts and memory mappings.

## Getting started

1. Clone the repository
```bash
git clone https://github.com/RyZeDZ/ChipMap.git
cd ChipMap
```

2. Install the requirements
```bash
pip install -r requirements.txt
```

3. Run the tool
```bash
python main.py
```
